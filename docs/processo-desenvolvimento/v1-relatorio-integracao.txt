A idéia é o que App Biblio faça o seguinte:
-cadastrar  literaturas
-listar as literaturas por ordem alfabetica
-permitir saida e retirada de quantidade de literaturas.

Durante a construção do aplicativo foi feito o uso prático de ferramentas e linguagens como:

-Linguagem C#, Entity Framework
-.NET 6.0
-Banco de Dados SQL Server / SSMS
-IDE MS Visual Studio (community)
-Angular 14
-Bootstrap: HTML/JS/CSS
